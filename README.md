# 🏠 AI Room Generator

A web app that generates interior room designs and restyles existing rooms using AI.

🚀 **Live Demo:** [room-generator-kasvii.streamlit.app](https://room-generator-kasvii.streamlit.app)

## Features

- **Style Transfer** — Upload a room photo and transform it into a different interior style
- **Room Generation** — Describe a room and generate a design from scratch

## Styles Available
Modern Minimalist · Bohemian · Industrial · Scandinavian · Mid-Century Modern · Rustic Farmhouse · Art Deco

## Stack
- Python, Streamlit
- Stable Diffusion XL via Hugging Face Inference API
- Deployed on Streamlit Community Cloud

## Note
This is a v1 prototype using the Hugging Face Inference API.
A v2 with a custom-trained VGG19 neural style transfer model is in development in a separate repo.
